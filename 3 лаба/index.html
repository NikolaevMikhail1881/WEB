Что такое FlexBox?

Flexbox - это целый модуль, содержащий в себе набор свойств и их значений как для родителя flex-контейнера так и для его дочерних элементов flex-элементов.

Основным преимуществом гибкой FLEX разметки является возможность изменять ширину/высоту пространство вокруг и порядок flex-элементов, для того чтобы наилучшим образом заполнить доступное пространство flex-контейнера

Колонки с flex-контейнерами
Гибкость распределения элементов
Flexbox позволяет легко управлять распределением элементов внутри контейнера, что делает его идеальным инструментом для создания адаптивного дизайна.
Удобное выравнивание
С помощью flexbox легко задавать выравнивание элементов как по горизонтали, так и по вертикали, что повышает гибкость в создании макетов.
Автоматическое увеличение размеров элементов
Flexbox автоматически изменяет размеры элементов так, чтобы они занимали всю доступную ширину контейнера, что делает его идеальным для построения сложных сеток.
Легкое изменение порядка элементов
С помощью свойства order можно менять порядок элементов без изменения HTML-структуры, что обеспечивает гибкость в организации контента на странице.
